## **שבוע 1 – יסודות FFmpeg**

**מטרה:** להבין את מבנה הפקודות, מושגים בסיסיים, ולהריץ פקודות ראשונות.

**נושאים:**

* מבנה הפקודה: `ffmpeg [input options] -i inputfile [output options] outputfile`
* מושגים בסיסיים: Container, Codec, Bitrate, Frame rate, Resolution
* הפעלת פקודה ראשונה

**תרגילים:**

1. התקן FFmpeg במחשב שלך ובדוק גרסה:

```bash
ffmpeg -version
```

2. המר קובץ MP4 ל-MKV.
3. חילץ את האודיו מקובץ וידאו ל-MP3.
4. בדוק את מידע הקובץ עם:

```bash
ffprobe input.mp4
```

---

## **שבוע 2 – עריכה בסיסית**

**מטרה:** חיתוך, מיזוג, שינוי פורמט בלי איבוד איכות.

**נושאים:**

* חיתוך וידאו ואודיו (`-ss`, `-to`, `-c copy`)
* מיזוג מספר קבצים (`concat`)
* שימוש בפקודות `-c copy` vs קידוד מחדש

**תרגילים:**

1. חתוך קובץ וידאו ל-30 שניות.
2. חבר שני קבצי וידאו יחד לקובץ אחד.
3. המר קובץ FLAC ל-MP3.
4. הרץ פקודה שמראה רק מסגרת מסוימת בקובץ וידאו.

---

## **שבוע 3 – עיבוד מתקדם**

**מטרה:** שינוי רזולוציה, frame rate, הוספת פילטרים.

**נושאים:**

* שינוי רזולוציה (`-vf scale`)
* שינוי frame rate (`-r`)
* חיבור פילטרים (`-vf`, `-af`)
* הוספת טקסט או watermark

**תרגילים:**

1. המר וידאו ל-720p.
2. שנה frame rate מ-30fps ל-60fps.
3. הוסף טקסט קטן בפינה של הוידאו.
4. המר אודיו למונו או סטריאו.

---

## **שבוע 4 – אוטומציה וסטרימינג**

**מטרה:** שילוב FFmpeg בסקריפטים, סטרימינג, אופטימיזציה.

**נושאים:**

* כתיבת סקריפטים Bash/PowerShell שמבצעים המרה אוטומטית
* הגדרת סטרימינג (RTMP/HTTP)
* אופטימיזציה של bitrate ואיכות
* בדיקה של לוגים ושגיאות FFmpeg

**תרגילים:**

1. כתוב סקריפט שממיר אוטומטית כל קובץ בוידאו בתיקייה ל-MP4.
2. הפק אודיו באיכות גבוהה אך עם גודל קטן.
3. נסה להפיק GIF מוידאו.
4. קרא לוג FFmpeg ופרש את השגיאות.
