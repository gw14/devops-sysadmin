* [ ] **Day 15-21: Python Scripting.** Study [Automate the Boring Stuff: Files](https://automatetheboringstuff.com/2e/chapter9/). Lab: Use Python `os` and `shutil` to automate file organization.
* [ ] **Day 22-28: CI/CD Foundations.** Study [Hello GitHub Actions](https://github.com/skills/hello-github-actions). Lab: Create a workflow that runs a script every time you `push` code.
